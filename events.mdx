## Event Discovery

Meetio's AI-powered event discovery helps you find relevant tech events worldwide. You receive personalized recommendations based on your profile, interests, and past attendance. Explore events by category, location, or skill level to expand your network.

## Browsing Events

<Steps>
  <Step title="Set Preferences" icon="settings">
    Update your profile with interests and skills to receive better event recommendations.
  </Step>
  <Step title="Search Events" icon="search">
    Use filters to find events by date, location, topic, or organizer.
  </Step>
  <Step title="Register" icon="check">

</Steps>

## Code Integration

<CodeGroup tabs="JavaScript,Python">
  ```javascript
const meetio = require('meetio-sdk');

const client = new meetio.Client({ apiKey: 'your-key' });

// Get recommended events
client.getRecommendedEvents({
  interests: ['javascript', 'ai'],
  location: 'remote'
}).then(events => {
  console.log('Recommended events:', events);
});

// Register for an event
client.registerForEvent('event-id-123')
  .then(() => console.log('Registered successfully'));
```
  ```python
from meetio import MeetioClient

client = MeetioClient(api_key='your-key')

# Get recommended events
events = client.get_recommended_events(
    interests=['python', 'machine-learning'],
    location='san-francisco'
)
print('Recommended events:', events)

# Register for an event
client.register_for_event('event-id-456')
print('Registered successfully')
```
</CodeGroup>
