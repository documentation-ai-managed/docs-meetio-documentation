## Community Building

Communities form the core of Meetio's platform, enabling students and developers to connect around shared interests. You create communities focused on specific technologies, locations, or career stages. Effective community management fosters engagement and growth.

## Code Examples

<CodeGroup tabs="JavaScript,Python">
  ```javascript
const meetio = require('meetio-sdk');

const client = new meetio.Client({ apiKey: 'your-key' });

// Create a new community
client.createCommunity({
  name: 'React Developers',
  description: 'Community for React enthusiasts',
  category: 'frontend',
  isPrivate: false
}).then(community => {
  console.log('Community created:', community.id);
});

// Get community members
client.getCommunityMembers(community.id).then(members => {
  console.log('Members:', members.length);
});
```
  ```python
from meetio import MeetioClient

client = MeetioClient(api_key='your-key')

# Create a new community
community = client.create_community(
    name='Python ML Group',
    description='Machine learning practitioners',
    category='ai',
    is_private=False
)
print(f'Community created: {community.id}')

# Get community members
members = client.get_community_members(community.id)
print(f'Members: {len(members)}')
```
</CodeGroup>
