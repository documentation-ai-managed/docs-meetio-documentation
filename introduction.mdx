## Quick Setup

<CodeGroup tabs="JavaScript,Python,cURL">
  ```javascript
const meetio = require('meetio-sdk');

const client = new meetio.Client({
  apiKey: 'your-api-key'
});

// Get your profile
client.getProfile().then(profile => {
  console.log('Welcome, ' + profile.name);
});
```
  ```python
from meetio import MeetioClient

client = MeetioClient(api_key='your-api-key')

# Get your profile
profile = client.get_profile()
print(f'Welcome, {profile.name}')
```
  ```bash
curl -X GET \
  'https://api.meetio.com/v1/profile' \
  -H 'Authorization: Bearer your-api-key'
```
</CodeGroup>
